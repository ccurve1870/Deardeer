penepy.util.getMaterials(*args) → List[material.Material][ソース]¶

penepy.materialPropertyList 中のMaterialを取得

パラメータ

*args (str) -- materialPropertyListのkey

戻り値

Material

戻り値の型

List[material.Material]

サンプル

i, W = penepy.getMaterials("iron", "WHA")

何個でも行ける

materialkey = ["iron", "WHA", "DU"]
materialList = penepy.getMaterials(*materialkey)
penepy.util.getTandP(mT: material.Material, mP: material.Material, L: float, D: float, Crh: float = 0.5, Ys: float = 0.0, ts: float = 0.0, th: float = 0.0) → List[Union[material.Target, material.Penetrator]][ソース]¶

Target と Penetrator を1行で取得するための関数

パラメータ

  • mT (material.Material) -- Targetに使うMaterial

  • mP (material.Material) -- Penetratorに使うMaterial

  • L (float) -- L[m]

  • D (float) -- D[m]

  • Crh (float, optional) -- Crh, by default 0.5

  • Ys (float, optional) -- 表面硬化領域の降伏強度[GPa], by default 0.

  • ts (float, optional) -- 完全に焼きが入った層の厚み[m], by default 0.

  • th (float, optional) -- 硬化層全体の厚み[m], by default 0.

[Target , Penetrator ]
